### 기능 View

회원관리 **accounts**

회원 팔로우 & 팔로우 취소

- `POST` http://127.0.0.1:8000/accounts/<int:user_pk>/follow/
- 로그인한 유저만 팔로우 기능을 사용할 수 있습니다.

### 화면 Template

회원 정보 페이지(프로필 페이지)

- `GET` http://127.0.0.1:8000/accounts/<int:user_pk>/
- 팔로우 목록
    
    해당 회원의 팔로우 목록
    
- 팔로잉 목록
    
    해당 회원의 팔로잉 목록
    

- 다른 회원의 프로필 페이지
    - 팔로우 버튼
        
        스스로를 팔로우 할 수 없습니다.
        
        로그인한 유저만 팔로우 기능을 사용할 수 있습니다.
        
        로그인 상태 / 팔로우 상태에 따라 다르게 표현 합니다.
